About Vibration Pattern Tester

Vibration Pattern Tester is a free online haptics pattern editor for developers and designers. Build custom vibration patterns with an interactive draggable timeline, preview instantly on web, and export ready-to-use code for Flutter vibration and Flutter haptics projects.

The tool supports the Web Vibration API and Web Audio API, making it easy to prototype haptic feedback for mobile apps, games, and progressive web apps. Whether you need a quick vibration pattern for a Flutter app or want to experiment with haptics patterns for Android and iOS, this tester helps you iterate in seconds.
